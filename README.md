# School Management System

A school management system built using Java, Spring Boot, Spring Security, JWT, MySQL, HTML, and JavaScript.

## Features
- User authentication with JWT
- Role-based access control
- Dashboards for Admin, Student, Teacher, and Accountant
- REST APIs for school operations
- MySQL database integration
- Secure backend using Spring Security

## Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT
- MySQL
- JPA / Hibernate
- HTML
- JavaScript
- Git
- Postman
